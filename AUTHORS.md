## Comment être listé ici

Lorsque vous contribuez au projet (code, documentation, design, etc.), vous êtes automatiquement reconnu comme contributeur. Votre nom apparaît dans :

1. **Ce fichier** — si contribution significative
2. **Git history** — accessible via `git log`
3. **GitHub Contributors graph** — visible sur la page du projet
4. **Releases notes** — pour les versions majeures

### Contribution via Pull Request

1. Ouvrez un PR
2. Incluez votre nom dans le commit message ou la description du PR
3. Nous vérifierons et fusionnerons votre contribution
4. Vous serez crédité automatiquement

### Contribution documentaire

Documentation, traductions, corrections, rapports de bugs : tous les efforts sont appréciés et crédités.
